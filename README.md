# Job Scraper

This project contains Python scripts to scrape job listings from various Nicaraguan job websites: Tecoloco, Computrabajo, and Encuentra24.

## Installation

1.  Clone the repository.
2.  Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

```bash
python tecoloco.py
    
python computrabajo.py

python encuentra24.py
```

## Dependencies

- `requests`
- `beautifulsoup4`
- `pandas`
- `streamlit`
